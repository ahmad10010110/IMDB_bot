# Movie Search Telegram Bot

This Telegram bot allows users to search for movies, explore their plots, and view trailers and posters. The bot is built using pure Python, utilizing the `requests` library to interact with the IMDb API and parse JSON responses without any additional libraries.

## Features

- **Search Movies**: Users can search for movies by title.
- **View Plot**: Get detailed plot summaries of movies.
- **Watch Trailers**: Access trailers for movies directly.
- **Posters**: Display movie posters from IMDb.

## Technologies Used

- **Python**: The programming language used to create the bot.
- **Requests**: For making HTTP requests to the IMDb API.
- **IMDb API**: To fetch movie data, including titles, plots, trailers, and posters.
